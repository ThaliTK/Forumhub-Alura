## 💻 Sobre o projeto

Este é o desafio final do programa **ONE (Oracle Next Education)** em parceria com a **Alura**. O ForumHub é uma API REST que replica o funcionamento de um fórum de discussões. 

O objetivo foi implementar um CRUD completo aplicando as melhores práticas do modelo REST, validações de dados e autenticação segura via tokens JWT.

## ⚙️ Funcionalidades

* **Criar um novo tópico:** Envio de título, mensagem, autor e curso.
* **Mostrar todos os tópicos:** Listagem paginada e organizada por data.
* **Mostrar um tópico específico:** Busca detalhada através do ID.
* **Atualizar um tópico:** Edição de informações de tópicos existentes.
* **Eliminar um tópico:** Remoção do banco de dados.

## 🔐 Segurança

* **Autenticação Stateless:** Utilização de tokens JWT para restringir o acesso.
* **Criptografia:** Armazenamento de senhas utilizando o algoritmo de hash BCrypt.

## 🛠️ Tecnologias

* Java 17
* Spring Boot 3
* Spring Data JPA
* Spring Security
* PostgreSQL
